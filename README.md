## 概要

このアプリはC#とMSSQLを利用した、日毎の業務を記録するシンプルなアプリです  
C#でアプリを作成する練習のために製作しました  
ユーザ作成やDB接続も基本的にアプリ内で操作が完結します

## 主な機能

- ユーザ作成
- MSSQLのDBを設定
- 日付毎に業務管理画面が作成され、そこに個々の作業項目を記録していく
- 作業項目には、作業の概要、予定の開始・終了時刻や実際の開始・終了時刻、業務完了の有無などが保存できる

## 開発環境

Visual Studio 2019  
C# 8.0  
Microsoft SQL Server 2019  
Microsoft SQL Management Studio
